Formation NetDevOps

Automatisation réseau de bout en bout — du prompt IA au déploiement physique

Hakim Oumeziane · Administrateur Réseaux & Télécoms

🎬 Vidéo en production

Objectifs

  • Comprendre l'approche NetDevOps et son positionnement par rapport au NetOps traditionnel
  • Construire un pipeline CI/CD complet pour la gestion de configurations réseau multi-constructeurs
  • Automatiser le déploiement via Python / Netmiko sur équipements physiques
  • Intégrer Claude AI comme moteur de génération et de validation de configurations
  • Appliquer les bonnes pratiques Git (branching, PR, review) dans un contexte réseau

Pipeline couvert

Claude AI ──► Claude Code ──► GitHub Actions ──► Netmiko ──► Équipements (stratégie) (génère+push) (validation CI) (Python) (déploiement)

Modules

Module 1 — Introduction NetDevOps

Concepts, différences avec NetOps/DevOps classique, cas d'usage réels, retour d'expérience terrain.

Module 2 — Git & GitHub pour les réseaux

Branching strategy, gestion de fichiers de configuration, PRs, révision de configs en code review.

Module 3 — Python & Netmiko

Connexion SSH multi-constructeurs (Cisco, Fortinet, HP/Aruba), envoi de commandes, récupération de sorties, gestion des erreurs.

Module 4 — GitHub Actions CI/CD

Validation de syntaxe, lint de configurations, tests de connectivité, auto-merge sur validation.

Module 5 — Claude AI comme co-pilote réseau

Génération de configurations depuis des prompts, validation sémantique, stratégie de prompts pour les équipements réseau.

Module 6 — Déploiement bout en bout

Pipeline complet de A à Z : prompt → génération → commit → CI → déploiement physique sur lab.
